# Installation und Wartung
## Ausgangssituation Anwendung läuft auf Server. Sie benötigt ein Betriebssystem, Bibliotheken (z.B. für DB-Zugriff) und evtl. Laufzeitumgebungen wie Java Runtime, Node.JS, Python, ...
## Installation - Betriebssystem, Bibliotheken und Laufzeitumgebungen - Anwendung selbst - Die notwendigen Schritte müssen exakt ausgeführt werden oder automatisiert werden, z.B. mit Ansible
